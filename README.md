## Features
- Gravity
- Realistic fuel usage
- Tractive effort simulation
- Dynamic max turn speed
- Mass and Power simulation

## Mod Support
Besides the steam engine this mod supports using motors from:
- Create: Diesel Generators
- Create: Electro Energetics
- Create: Crafts & Additions

## Q&A

How to make trains run?

> Put a steam engine block onto the locomotive

How to make trains run faster?

> Put more engines onto the locomotive
